# Agentic Shopping Agent

Goal:
An AI shopping assistant that finds and ranks products based on:
- relevance
- price
- quality
- reviews
- specifications
- user intent

Current Problem:
The item selection logic is weak.
The system often:
- picks the first matching product
- overweights name similarity
- ignores better alternatives
- fails at semantic ranking

Desired Improvements:
- smarter ranking
- semantic understanding
- weighted scoring
- better filtering
- robust architecture
- production-ready reliability

Constraints:
- minimize API costs
- fast inference
- modular architecture
- maintainable code
